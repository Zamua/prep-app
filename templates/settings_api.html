How to use

Set the token as a Bearer credential on the Authorization header: 

curl -H "Authorization: Bearer prep_pat_..." \
  https://prepcards.app/api/v1/decks

Full endpoint reference + try-it-now console live at /docs (Swagger UI) or /redoc. Raw schema at /openapi.json.
