Connect via MCP

Your AI agent talks to prep over HTTP via the Model Context Protocol. Three steps:

  1. Mint a token on the API tokens page. The full token is shown once; copy it before leaving.
  2. Add prep as an MCP server in your client.
    Claude Code (CLI) 
    claude mcp add --transport http prep \
  https://prepcards.app/mcp \
  --header "Authorization: Bearer prep_pat_..."
    Claude Desktop

    Edit claude_desktop_config.json (Settings → Developer → Edit config) and add: 

    {
  "mcpServers": {
    "prep": {
      "url": "https://prepcards.app/mcp",
      "headers": {
        "Authorization": "Bearer prep_pat_..."
      }
    }
  }
}

    Restart Claude Desktop after saving.

    Other MCP-aware clients

    Point the client at https://prepcards.app/mcp (HTTP transport, JSON-RPC 2.0). Pass the token as Authorization: Bearer prep_pat_….

  3. Use it. Your agent gets a tool catalog covering nearly every read/write on your decks + cards — imports, exports, rename, pin, suspend, the lot. Full reference is in /llms.txt (the agent-discovery manifest) and /docs (Swagger UI).
