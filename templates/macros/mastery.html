{# Mastery bar macro — green/red/empty progress track. Used in two places: the deck page header (lg variant, default) and the index deck card (mini variant). Same markup; size + fill alpha differ via the `--mini` modifier on .mastery-bar. Parameters: - mastered: int — count of mastered cards - wrong: int — count of cards last answered wrong - total: int — total cards in the deck - mini: bool — apply the mini-size modifier (smaller height, softer fills) #} {% macro bar(mastered, wrong, total, mini=False) %} {%- set _t = total or 1 -%} {%- set _mp = (100 * mastered / _t) -%} {%- set _wp = (100 * wrong / _t) -%} {% endmacro %}
